# 台灣口罩地圖

透過取得健康保險開放資料服務，來獲取口罩數量及特約醫院相關資料

## 功能:
* 提供地圖導航
* 電話撥打
* 即時口罩數量顯示
* 顯示醫院相關資訊
* 資料查詢
* 地圖顯示
